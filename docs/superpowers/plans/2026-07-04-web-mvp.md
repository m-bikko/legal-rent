# Web MVP «RentLegal KZ» Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Рабочий web MVP платформы аренды: телефон+OTP auth (dev-заглушка), лента объявлений с фильтрами/лайками, детали объекта, верификация арендодателя, публикация объектов, договор с подписанием и статусами.

**Architecture:** Монорепо pnpm+Turborepo. Все данные — через Next.js Route Handlers с Supabase service-role (браузер к Supabase не ходит). Сессия — httpOnly JWT cookie (jose). Бизнес-логика и типы — в `packages/core`, словари — в `packages/i18n`.

**Tech Stack:** Next.js 15 (App Router, React 19), ant-design v5 (+ v5-patch-for-react-19, nextjs-registry), Tailwind v4, @tanstack/react-query v5, next-intl v4, zod, jose, Supabase (Postgres + Storage), Jest + RTL, pg (для миграций).

**Спека:** `docs/superpowers/specs/2026-07-04-web-mvp-design.md`

**Правило пользователя:** перед написанием кода antd/Tailwind — сверяться с документацией через context7 MCP (`resolve-library-id` → `query-docs`).

```ts
/** Нормализует ввод в E.164 (+77XXXXXXXXX) или null. Мобильные КЗ: +7 7XX. */
export const normalizeKzPhone = (raw: string): string | null => {
  const digits = raw.replace(/\D/g, "");
  let ten: string;
  if (digits.length === 11 && (digits.startsWith("8") || digits.startsWith("7"))) ten = digits.slice(1);
  else if (digits.length === 10) ten = digits;
  else return null;
  return /^7\d{9}$/.test(ten) ? `+7${ten}` : null;
};

export const formatKzPhone = (e164: string): string =>
  e164.replace(/^\+7(\d{3})(\d{3})(\d{2})(\d{2})$/, "+7 $1 $2 $3 $4");
```

- [ ] **Step 2:** FAIL. **Step 3:** Реализация (чистые функции, `node:crypto` sha256; константы `OTP_TTL_MS = 5*60_000`, `OTP_MAX_ATTEMPTS = 5`, `OTP_RESEND_COOLDOWN_MS = 60_000`):

```ts
import { createHash, randomInt } from "node:crypto";
export const OTP_TTL_MS = 5 * 60_000;
export const OTP_MAX_ATTEMPTS = 5;
export const OTP_RESEND_COOLDOWN_MS = 60_000;

export const generateOtpCode = (): string => randomInt(0, 1_000_000).toString().padStart(6, "0");
export const hashOtp = (code: string, phone: string, secret: string): string =>
  createHash("sha256").update(`${code}:${phone}:${secret}`).digest("hex");

export type OtpCheckResult = "valid" | "invalid" | "expired" | "consumed" | "too_many_attempts";
export const checkOtp = (row: { codeHash: string; expiresAt: Date; attempts: number; consumed: boolean },
  code: string, phone: string, secret: string, now: Date): OtpCheckResult => {
  if (row.consumed) return "consumed";
  if (row.attempts >= OTP_MAX_ATTEMPTS) return "too_many_attempts";
  if (now > row.expiresAt) return "expired";
  return hashOtp(code, phone, secret) === row.codeHash ? "valid" : "invalid";
};
```

- [ ] **Step 4:** PASS. **Step 5:** Commit `feat(core): otp logic`.
